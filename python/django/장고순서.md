#장고로 프로젝트 생성하기의 기본과정

1. django-admin startproject 프로젝트생성

2. python manage.py startapp 앱 생성

3. 생성한 앱을 메인폴더의 settings.py에 등록

4. settings.py의 자잘한 설정등록 <br>
Time_Zone = 'Asia/Seoul' (시간대변환) <br>
USE_TZ = False (DB에 저장되는 시간도 한국으로 저장함) <br>
LANGUAGE_CODE = 'ko-kr' (한국어 설정: admin페이지등)

5. Model 생성 (unique, foriegn) <br>
   admin페이지에 등록 @admin.register(모델) <br>
   원하는 title이 표기되도록
   
   ```python
   def __str__(self):
       return self.title
   ```

6. 메인폴더의 urls.py 에서 app의 urls.py로 연결 (include 사용)

7. 연결된 urls.py
- from django.urls import path
- from . import views
- path('[]', views.[], name='[]')
